# Идея по разработке LLM-ассистента

## Общая концепция

LLM-ассистент должен быть выполнен в виде Telegram-бота.

## Основная задача

Основной задачей бота является вести диалог с пользователем и отвечать на его вопросы с ролью профессионального консультанта банка, заданной в системном промпте LLM.
